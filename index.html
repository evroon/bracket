Free and open source tournament scheduling system

Build tournament setups, add teams, schedule matches, keep track of scores and present ranking live to the public.

GitHub

Go to the GitHub repository to star or fork Bracket, create issues/PRs or start discussions.

Features

Open-source and free

Bracket is fully open source and free to use, licensed under the AGPL-3.0 license.

Flexible

Bracket supports the standard tournament types, teams can be added/changed during the tournament and new matches can be scheduled dynamically.

Easy to use

The UI is meant to be easy to use while providing maximum flexibility.

Self-hosted

You are free to host it yourself. Setup is easy; either run it in Docker or run it the natively on the host. The only external dependency is a PostgreSQL database.
